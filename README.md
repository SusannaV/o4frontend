Kommentoin App.jsx:ään ja Tuotekortti.jsx:ään mitä ne tekee. Generoin tähän kansioon json-tiedoston, jossa on lista tuotteista, joka nyt tuodaan vaan suoraan kansiosta App.jsx:ään. Myöhemmin tämä tiedosto poistetaan tästä projektista, koska backend tarjoaa meille oikean listan (joko tarjoukset-, tietokoneet- tai oheislaitteet-listan).

Jätin .css-tiedoston tänne olemaan, että on jotain muotoilua valmiina, muokataan sitä sit rankasti myöhemmin.
